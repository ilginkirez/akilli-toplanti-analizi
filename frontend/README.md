# Meeting Analyzer Frontend

Bu uygulama, projenin ana web arayuzudur. React, Vite ve LiveKit istemcisi ile
toplanti yonetimi, giris akisleri ve canli gorusme deneyimini tek yerde toplar.

## Gelistirme

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Klasorler

- `src/app/pages`: ekranlar ve route hedefleri
- `src/app/components`: ortak arayuz bilesenleri
- `src/app/livekit-meeting`: canli toplanti deneyimi
- `src/app/auth`: oturum yonetimi
- `src/app/meetings`: toplanti store ve akislari
